# Presentation Design Styles

A library of 30 named design styles for slide presentations. Each style defines a complete visual system: color palette, typography pairing, layout principles, and usage guidance.

Full specifications for all 30 styles live in `references/styles.md`. This skill provides the selection framework and a quick-reference overview. Do not copy style specs inline — always direct users to the reference file for the full palette and layout details.

## Style Categories

### Corporate / Professional (Styles 01–05)

Formal, authoritative palettes designed for business contexts where credibility and clarity take precedence over visual personality.

- **Executive Suite** (01) — Deep navy + antique gold. Board-level strategy, investor briefings, C-suite decks.
- **Corporate Blue** (02) — Mid-range blue + amber. Stakeholder communications, status reports, cross-department proposals.
- **Professional Gray** (03) — Charcoal + signal blue. Data analytics, internal reporting, consulting deliverables.
- **Enterprise Dark** (04) — Deep slate + sky blue. Technology product presentations, R&D briefings, evening venue talks.
- **Business Formal** (05) — Black + deep red. Legal, financial, regulatory, and compliance presentations.

### Modern / Minimal (Styles 06–10)

Clean, reduction-focused aesthetics that let content speak. Generous whitespace, restrained color use, systematic typography.

- **Clean White** (06) — Pure white + single focus blue. Product launches, keynote-style narrative, design portfolio showcases.
- **Nordic Minimal** (07) — Warm cream + forest sage + clay. Wellness, sustainability, non-profit, creative agency decks.
- **Swiss Grid** (08) — Signal red + black. Consulting, government, design conference talks requiring strict grid discipline.
- **Flat Modern** (09) — Violet + coral + teal. Startup pitches, product demos, app onboarding, hackathon talks.
- **Monochrome** (10) — Single-hue palette in tints and shades. Brand identity presentations, portfolio reviews, academic research.

### Creative / Bold (Styles 11–15)

High-energy, high-saturation designs for contexts where memorability outweighs formality.

- **Vibrant Pop** (11) — Hot coral + electric yellow + cyan. Consumer brands, youth campaigns, creative agency pitches.
- **Bold Contrast** (12) — Vivid orange + near-black. Product launches, brand manifestos, keynote announcements.
- **Electric Neon** (13) — Neon green + electric violet + deep space. AI product launches, cybersecurity, deep tech, gaming industry.
- **Gradient Flow** (14) — Hot pink + violet + cyan multi-stop gradients. Brand launches, creative portfolios, award ceremonies.
- **Dynamic Energy** (15) — Power red + energy orange + near-black. Sports, fitness brands, high-growth startups, marketing reveals.

### Academic / Research (Styles 16–20)

Serif-forward, evidence-first aesthetics aligned with scholarly publishing conventions.

- **Academic Classic** (16) — Scholar navy + muted gold + parchment. University lectures, conference talks, thesis defenses.
- **Research Paper** (17) — Research blue + highlight red. Academic conferences (ICML, NeurIPS, AGU), grant panels, lab reviews.
- **Scientific Data** (18) — Okabe-Ito colorblind-safe palette. Data analysis, regulatory submissions, evidence-based policy talks.
- **Scholarly Dark** (19) — Antique gold + warm parchment + deep brown. Distinguished lectures, humanities, literary events.
- **Publication Ready** (20) — Midnight blue + cinnabar red + white. Conference papers with distributed handouts, journal clubs.

### Warm / Organic (Styles 21–25)

Earth-toned, tactile palettes evoking natural materials, craftsmanship, and organic warmth.

- **Warm Earth** (21) — Terracotta + ochre + sandy cream. Agriculture, food, sustainability, artisan business decks.
- **Natural Wood** (22) — Walnut brown + honey amber + light oak. Woodworking, furniture, craft beverage, sustainable architecture.
- **Sage Garden** (23) — Sage green + dusty rose + linen white. Wellness, botanical, health and nutrition brands.
- **Sunset Glow** (24) — Coral orange + warm amber + rose pink. Hospitality, travel, wedding industry, inspirational keynotes.
- **Desert Sand** (25) — Canyon terracotta + pale buff + dark clay. Real estate, outdoor recreation, pottery, Southwestern cultural institutions.

### Dark / Luxury (Styles 26–30)

Premium dark-surface aesthetics communicating exclusivity, technical authority, or industrial precision.

- **Midnight Premium** (26) — Antique gold + midnight navy. Luxury brand launches, private equity, fashion house presentations.
- **Dark Luxury** (27) — Rose gold + matte black. Premium tech products, high-end automotive, luxury cosmetics.
- **Obsidian** (28) — White on pure black, no accent. Ultra-premium brand philosophy, architecture portfolios, brand manifestos.
- **Deep Ocean** (29) — Cerulean + ocean blue + deep navy. Marine research, oceanography, water infrastructure, environmental science.
- **Carbon Black** (30) — Safety orange + carbon black. Automotive, aerospace, engineering, advanced manufacturing.

## How to Apply a Style

### Step 1 — Select a Style

Identify the most appropriate style using the Quick Style Selection Guide above. When in doubt, consider:

- **Audience**: Executive → Corporate/Dark styles. Technical → Enterprise/Scientific. Creative → Bold/Warm styles.
- **Tone**: Formal → 01–05, 16–20. Playful → 09, 11. Luxury → 26–28. Organic → 21–25.
- **Venue**: Boardroom → 01. Evening venue → 04. Lecture hall → 16–17. Event stage → 12–15.

Consult `references/styles.md` for the selected style's complete color palette (7 values), typography pairing (heading / body / caption), layout principles, and anti-patterns.

### Step 2 — Apply Color Using Aesthetic Presets

For HTML-based slides, map the selected style's palette to the 8 CSS custom properties defined in `references/aesthetic-presets.md`. Five named presets (Executive, Academic, Technical, Product, Editorial) provide ready-to-use CSS blocks that can be applied globally or scoped per-slide.

To use a preset:
- **Global**: Set variables on `:root`.
- **Scoped**: Apply a `.theme-*` class to individual slide elements.
- **Programmatic**: Use `data-theme` attributes for agent-generated decks.

For PPTX generation via `python-pptx + SlideFactory`, map palette values to the `Theme` object's `primary`, `accent`, and `font_name` parameters.

### Step 3 — Apply Typography

Each style in `references/styles.md` specifies three typeface roles:

- **Heading**: Display or serif typeface — largest size, highest weight.
- **Body**: Readable sans-serif or serif — 15–18pt effective at projector distance.
- **Caption**: Monospace or light variant — for labels, citations, and metadata.

Limit each deck to two typeface families maximum. Use system font stacks (`ui-serif`, `ui-sans-serif`, `ui-monospace`) for portability across rendering environments.

### Step 4 — Follow the Design Methodology

The 5-step slide design methodology in `references/slide-patterns.md` governs how to apply any style correctly:

1. **Define the Message** — Every slide title states a conclusion, not a topic.
2. **Choose the Style** — Match aesthetic to audience and venue (this skill).
3. **Structure the Layout** — Apply one of 6 primary layout archetypes.
4. **Apply Color and Typography** — Use CSS custom properties; accent color on one element per slide.
5. **Validate and Refine** — Content, visual, and narrative checklists.

Never apply a style without following this methodology. The style provides the visual language; the methodology ensures it communicates effectively.

### Anti-Patterns (Universal)

These apply regardless of which of the 30 styles is selected:

- Never place a decorative horizontal rule or accent line directly under a heading — this is a recognized AI-generated content marker.
- Never hard-code colors outside the style's defined 7-value palette.
- Never mix two styles within a single deck.
- Never use accent color on more than one element per slide.
- Never exceed two typeface families in a deck.
